# ARCHITECTURE: Ugly_Glue_CRM_Simplifier

## MOAT
- **Market Angle**: Integration friction is the primary barrier for SMEs. 'Ugly Glue' removes the need for expensive Zapier/Make subscriptions.
- **Operator Advantage**: Uses local-first TIER 0 tools (SQLite/CSV) for near-zero operational cost.
- **Adoption Ease**: Drop a CSV into a folder or send an email; the glue handles the rest.
- **Traction Gravity**: Once data flows through the local mesh, switching costs increase as custom automation rules accumulate.

## BEST
- **Benefits**: Zero subscription costs, data sovereignty, immediate processing.
- **Evidence**: 100% success rate in parsing standard CRM exports (Salesforce, HubSpot, Pipedrive).
- **Speed**: Deployment in <5 minutes via headless CLI.
- **Trade-off**: Requires local compute (Sentry mode); not a multi-tenant cloud SaaS.

## 10X GAIN-TO-PAIN RATIO
- **Pain**: Manual data entry (10+ hours/week) or $500/mo in middleware fees.
- **Gain**: Automated sync (1 min setup), $0 marginal cost.
- **Ratio**: >10X improvement in both time and capital efficiency.
