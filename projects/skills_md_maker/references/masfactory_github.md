## Overview
MASFactory is a graph-centric framework for orchestrating Multi-Agent Systems with Vibe Graphing.

## Key Features
- Vibe Graphing: intent → graph → compile → execute
- Graph-style composition with Node/Edge
- Visualization & observability
- Human-in-the-loop interaction
