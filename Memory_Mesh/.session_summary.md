- **Leads Summary:** Total, Qualified, Critical, Pending

- **Reports:** One-click Telegram/Email/Console

### Keyboard Shortcuts (TUI)
| Key | Action |
|-----|--------|
| `r` | Refresh all |
| `t` | Send Telegram report |
| `e` | Send Email report |
| `p` | Poll Gmail |
| `q` | Quit |

### Significant Update Triggers
Reports auto-send when:
- Critical leads (score 9+) detected
- Qualified leads (score 7+) detected
- New enquiries received
- System health issues (RAM >90%, Swap >80%, Ollama down)
