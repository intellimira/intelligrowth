# Memory Mesh

> MIRA's persistent knowledge storage - The Weave's physical layer

## Structure

```
Memory_Mesh/
├── zettels/           # Atomic knowledge units
├── protocals/         # Core operational procedures
├── case_studies/      # Documented problem-solving episodes
└── sparks/            # Quick-access memory fragments
```

## Usage

- **active_recall** scans this folder for context
- **context_compactor** consolidates content here
- **topic_sealer** creates zettels from reasoning
