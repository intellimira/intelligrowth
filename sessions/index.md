## Session Naming Convention

Format: `ses_YYYYMMDDHHMM_project-name.md`

## Creating New Sessions

When starting work on a task:

1. **Check existing projects** - Does this fit an existing project?
2. **Query user if unsure** - "This fits project X. Continue or new?"
3. **Create session log** in project `docs/` folder
4. **Mirror to central** `sessions/` folder
5. **Update this index** with new entry
