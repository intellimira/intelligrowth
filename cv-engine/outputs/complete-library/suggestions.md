## 5.1 Application Strategy

### Volume vs. Quality
- **Recommended:** 60% quality (targeted), 40% volume
- Target: 10-15 applications per week
- Response rate expectation: 10-15%

### Application Workflow

1. **Research (10 min)**
   - Read job description fully
   - Identify 5-7 keyword matches
   - Note hiring manager name if possible

2. **Tailor CV (5-10 min)**
   - Reorder achievements to match
   - Add missing keywords naturally
   - Keep original wording (don't fabricate)

3. **Apply (2 min)**
   - Customized cover letter: 3 sentences max
   - Submit via ATS + LinkedIn Easy Apply

4. **Follow-up (Day 7)**
   - LinkedIn connection to hiring manager
   - Brief, non-desperate message
