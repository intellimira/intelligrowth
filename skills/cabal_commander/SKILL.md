# Cabal Commander

The strategic management layer for ACCT v1.3. It delegates tasks via `cabal_spawner` and unifies their output into a single mission success report.

## Trigger Phrases
- "Coordinate these tasks..."
- "Manage the sub-nodes for..."
- "Synthesize the results of the mission..."

## Usage
```python
python3 scripts/core.py
```
