# Topic Sealer

The "Synaptic Pruning" mechanism for ACCT. It moves historical turns to the sealed vault and generates a semantic summary.

## Trigger Phrases
- "Seal the current topic..."
- "Consolidate these turns into EXP..."
- "Finish the sub-quest and archive logs..."

## Usage
```python
python3 scripts/core.py
```
