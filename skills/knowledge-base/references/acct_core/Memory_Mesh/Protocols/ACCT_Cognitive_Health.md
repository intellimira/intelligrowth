*   **Action:** If a Zettel is found with zero incoming/outgoing links, it is flagged as "Isolated Context" for refactoring.

### Tier 3: Strategic Alignment (L2)

*   **Action:** Flag any tasks that deviate from "Recursive Intelligence" or "Contextual Sovereignty."

### Tier 4: Master Skill Fidelity (L3)
*   **Verification:** Perform automated health checks on the Skill Vault.
*   **Action:** Run a "Watchdog Pulse" on each Master Skill to ensure logic-streams are not failing or producing high-latency results.

## 3. The Health Score (A-Rank to E-Rank):
ACCT will generate a **Cognitive Health Score** at the start of every session:
*   **100-90% (S-Rank):** Perfect mesh coherence. High-speed UTO execution.
*   **89-75% (A-Rank):** Strong coherence. Minor isolated context found.
*   **74-50% (B-Rank):** Moderate fragmentation. Nodal imbalances detected.
*   **<50% (E-Rank):** Systemic decay. Immediate Context Restoration required.
