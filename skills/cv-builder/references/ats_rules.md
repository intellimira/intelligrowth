# ATS Optimization Rules

## Page Limits

| System | Max Pages |
|--------|-----------|
| Most ATS | 2-3 pages |
| Strict systems | 2 pages |
| Flexible | 3-5 pages |

## Keyword Guidelines

### Must Include
- Exact job title keywords
- Skills from job description
- Industry-specific terms

### Density
- Aim for 5-7% keyword density
- Scatter throughout, not clustered

## Formatting Rules

### DO
- Use standard section headers
- Keep simple bullet points
- Use consistent date formats (MM/YYYY)
- Plain text or simple tables

### DON'T
- Headers in all caps
- Complex tables with merged cells
- Graphics, icons, images
- Columns (confuses most ATS)
- Text boxes

## Section Order (Recommended)

1. Contact Info
2. Summary/Profile
3. Skills
4. Experience
5. Education
6. Certifications

## Common ATS Keywords

### Technical
- AWS, Azure, Python, SQL
- ETL, Data Lake, Pipeline
- Agile, Scrum, JIRA

### Business
- Stakeholder Management
- Requirements Gathering
- ROI, Cost Savings

## Testing

Run through:
- JobScan
- Resume.io
- TopResume
